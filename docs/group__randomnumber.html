integer(i8) function rng_integer_pos ( integer(i8), intent(in)  n)

[0, ), [0, ]

This function returns a random integer from 0 to N-1 inclusive by scaling down and/or discarding samples. All integers in the range [0,N-1] are produced with equal probability. This function is designed for sampling from ranges smaller than the range of the underlying generator. To use the full range use random_int() with no argument

Parameters
[in]nUper limit of the desired range
Returns
integer random number generated Examples:
print *, rng_int() ! Random integer in the range [-2^63, 2^62+1]
print *, rng_int(6) ! random integer in the range [0,5]
