Detailed Description

Functions returning a variate x with normal probability.

Valid uses:

USE numfor, only rng_normal
x = rng_normal2()
x = rng_normal2(scale)
x = rng_normal2(loc, scale)

This version is faster than rng_normal() but is not thread-safe
